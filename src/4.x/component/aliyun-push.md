# Aliyun 推送



## 安装

```
composer require poppy/aliyun-push 4.0.*
```

更新你的依赖包  `composer update`  或者全新安装  `composer install` 。

## 注意

在使用推送的时候如果发送的是消息, 则 extras 会放到 body 中来进行发送
